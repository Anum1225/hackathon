# Project Context

## Project Identity
- **Name**: Physical AI & Humanoid Robotics Textbook
- **Tagline**: Bridging the gap between the digital brain and the physical body
- **Organization**: Panaversity
- **Website**: https://panaversity.org
- **Repository**: https://github.com/panaversity/physical-ai-book

## Technical Stack
- **Static Site Generator**: Docusaurus v3.9.2
- **Core Library**: React v19.0.0
- **Language**: TypeScript v5.6
- **Build Tool**: Webpack (via Docusaurus)
- **Styling**: Custom CSS, Infima (Docusaurus default)

## Key Directories
- `/docs`: Markdown content for the textbook modules.
- `/src/components`: React components (HomepageFeatures, Chatbot, etc.).
- `/src/pages`: Landing page and other standalone pages.
- `/src/css`: Global styles (`custom.css`).

## Current State
- The project is a comprehensive educational resource for Physical AI.
- Features a modern, dark-themed UI with "Glassmorphism" design elements.
- Includes interactive components like a Chatbot (under development).
